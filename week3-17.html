The Internet

A network is a group of connected, communicating devices such as computers andprinters. An internet (note the lowercasei) is two or more networks that can communicate with each other.The most notable internet is called the Intercomposed of hundreds of thousands of interconnected networks.

The protocol is the client-server application program used to retrieve the document. Many different protocols can retrieve a document; among them are Gopher, FTP, HTTP, News, and TELNET. The most common today is HTTP. The host is the domain name of the computer on which the information is located. Web pages are usually stored in computers, and computers are given domain name aliases that usually begin with the characters "www". This is not mandatory, however, as the host can have any domain name.
